## 未附带字体

### Windows 自带字体（如 Segoe UI / Segoe UI Variable / Microsoft YaHei UI）

没有把这些微软随 Windows 提供的系统字体直接附带进公开 Release。

原因是微软官方 FAQ 明确写了：除文档嵌入等特殊情况外，`Windows supplied fonts` 不能重新分发，也不能复制到其他电脑或服务器。

- Microsoft Learn： [Font redistribution FAQ](https://learn.microsoft.com/en-my/typography/fonts/font-faq)
- Microsoft Learn： [Segoe UI font family](https://learn.microsoft.com/en-us/typography/font-list/segoe-ui)

因此软件里新增的“恢复 Windows 默认”采用的是：

- 恢复注册表里的字体映射和渲染参数
- 如果系统字体文件本身损坏或被删，再调用 `DISM /RestoreHealth` 和 `sfc /scannow`

### MiSans

没有把 `MiSans` 附带进发布包。

原因是它的官方授权明确限制对字体软件及其副本的再次分发，因此不适合直接打进公开 Release。
