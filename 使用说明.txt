Windows字体调谐器 使用说明

这版软件已经内置三套字体包：
- HarmonyOS Sans SC
- Source Han Sans CN
- Sarasa UI SC

推荐使用顺序：
1. 右键 WindowsFontTuner.exe，选择“以管理员身份运行”。
2. 在软件里选择一个预设。
3. 先点“安装预设字体”。
4. 再点“应用当前预设”。
5. 如果不满意，可以点“恢复最近备份”。
6. 想回到微软原生方案，可以点“恢复 Windows 默认”。
7. 如果系统自带字体文件被删了，再点“修复系统字体”。

更新说明：
- 软件启动后会自动检查 GitHub Release。
- 也可以手动点击“检查更新”。
- 发现新版本后，可以直接打开项目页或下载页。

三套预设建议：
1. HarmonyOS 统一预设
   更现代、更干净，适合想要统一感的人。
2. 思源黑体 CN 统一预设
   最稳妥、最中性，适合作为长期默认方案。
3. 更纱 UI SC 统一预设
   字形更有存在感，适合觉得 Windows 默认太细的人。

注意事项：
- 本工具会修改系统字体映射和渲染参数，请务必以管理员身份运行。
- 恢复最近备份主要回退映射与渲染设置，不会自动卸载已经安装的字体文件。
- “恢复 Windows 默认”会清除本工具写入的替换项，并恢复常见默认渲染参数。
- 软件不会公开附带微软自带字体文件；如果这些字体被误删，请使用“修复系统字体”调用 Windows 官方修复。
- Windows 11 不同界面对字体设置的响应可能不完全一致。
